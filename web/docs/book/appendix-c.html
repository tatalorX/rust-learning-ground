Exercise 015: Ownership Basics
Key insight: Understand move semantics. When you assign a String to another variable, ownership moves.
fn main() {
    let s1 = String::from("hello");
    let s2 = s1;  // s1 is MOVED to s2
    // println!("{}", s1);  // ERROR: s1 no longer valid
    println!("{}", s2);  // OK
}
Exercise 020: Borrowing
Remember: Either one mutable reference OR any number of immutable references.
fn main() {
    let mut s = String::from("hello");
    
    let r1 = &s;  // OK: immutable borrow
    let r2 = &s;  // OK: multiple immutable borrows
    println!("{} {}", r1, r2);
    
    let r3 = &mut s;  // OK: r1 and r2 no longer used
    r3.push_str(" world");
}

Exercise 055: Iterator Basics
Iterators are lazyâ€”nothing happens until you call a consuming method.
fn main() {
    let sum: i32 = (1..=10)
        .filter(|x| x % 2 == 0)  // Keep even numbers
        .map(|x| x * x)           // Square them
        .sum();                   // Sum them up
    
    println!("Sum of squares of even numbers 1-10: {}", sum);

Exercise 066: Closures
Closures capture their environment. The compiler determines how (by reference, mutable reference, or by value).
fn main() {
    let multiplier = 3;
    
    let multiply = |x| x * multiplier;  // Captures multiplier by reference
    
    println!("3 * 5 = {}", multiply(5));
}

Chapter 12-18: Mastery (Exercises 130-320)

Exercise 280: Thread Spawn
Use the move keyword to transfer ownership to the thread.
use std::thread;

fn main() {
    let data = vec![1, 2, 3];
    
    let handle = thread::spawn(move || {
        println!("Data in thread: {:?}", data);
    });
    
    handle.join().unwrap();
}
Exercise 300: Mutex
Always wrap Mutex in Arc when sharing between threads. The lock is automatically released when the guard is dropped.
use std::sync::{Arc, Mutex};
use std::thread;

fn main() {
    let counter = Arc::new(Mutex::new(0));
    let mut handles = vec![];
    
    for _ in 0..10 {
        let counter = Arc::clone(&counter);
        let handle = thread::spawn(move || {
            let mut num = counter.lock().unwrap();
            *num += 1;
        });
        handles.push(handle);
    }
    
    for h in handles { h.join().unwrap(); }
    
    println!("Result: {}", *counter.lock().unwrap());
}
